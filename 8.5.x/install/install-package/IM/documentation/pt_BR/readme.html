Suporte de Software do IBM Rational

Para obter informações de contato e diretrizes ou materiais de referência que serão necessários ao solicitar suporte, leia o IBM Software Support Handbook.

Para obter FAQs, listas de problemas conhecidos e correções, além de outras informações sobre suporte, visite o IBM Installation Manager portal Support.

Para obter notícias, eventos e outras informações do produto, visite o IBM Rational Web site do Software.

Antes de entrar em contato com o Suporte de Software do IBM Rational, reúna as informações de plano de fundo que serão necessárias para descrever o problema. Ao descrever um problema para um especialista em suporte de software IBM, seja tão específico quanto possível e inclua todas as informações relevantes de plano de fundo, para que o especialista possa ajudá-lo a resolver o problema de forma eficiente. Para economizar tempo, saiba as respostas a estas perguntas:

  • Quais versões de software você estava executando quando o problema ocorreu?
  • Você tem logs, rastreios ou mensagens que estão relacionados ao problema?
  • Você consegue reproduzir o problema? Se sim, quais etapas você executa para reproduzi-lo?
  • Há uma solução alternativa para o problema?
  • Se sim, esteja preparado para descrever a solução alternativa.
