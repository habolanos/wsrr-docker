Corrigindo o erro
=================
A barra de lançamento suporta os seguintes navegadores: 

Mozilla Firefox versão 3.5 e posterior
Microsoft(R) Internet Explorer, versão 6.0 e posterior (apenas plataformas Microsoft Windows(R))

Para corrigir o erro que impede a inicialização da barra de ativação, siga estas etapas: 

1.  Instale ou atualize para um navegador suportado. 

    Nas plataformas Linux(R) e UNIX(R), exporte o local do navegador suportado. 
    Por exemplo, digite o seguinte na linha de comandos:
    export BROWSER=/usr/bin/firefox

2.  Se o DVD do produto não estiver acessível, insira o DVD em uma unidade de DVD. 

3.  Em plataformas Linux e UNIX, monte a unidade, se necessário.

4.  Reinicie a barra de ativação
	Em plataformas Windows, digite o seguinte na linha de comando:
		launchpad.exe
	Em plataformas Linux e UNIX, digite o seguinte na linha de comando:
		./launchpad.sh
