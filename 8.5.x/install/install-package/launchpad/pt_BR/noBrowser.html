Corrigindo o Erro

A Barra de Ativação suporta os seguintes navegadores:

  • Mozilla Firefox versão 3.5 e posterior
  • Microsoft® Internet Explorer, versão 6.0 e mais recentes (somente plataformas Microsoft Windows®)

Para corrigir o erro que impede a inicialização da barra de ativação, siga estas etapas:

  1. Instale ou atualize para um navegador suportado.

    Nas plataformas Linux® e UNIX®, exporte o local do navegador suportado.
    Por exemplo, digite o seguinte na linha de comandos:
    export BROWSER=/usr/bin/firefox

  2. Se o DVD do produto não estiver acessível, insira o DVD em uma unidade de DVD.
  3. Em plataformas Linux e UNIX, monte a unidade, se necessário.
  4. Reiniciar a barra de ativação
    Em plataformas Windows, digite o seguinte na linha de comandos:
    launchpad.exe
    Em plataformas Linux e UNIX, digite o seguinte na linha de comandos:
    ./launchpad.sh
