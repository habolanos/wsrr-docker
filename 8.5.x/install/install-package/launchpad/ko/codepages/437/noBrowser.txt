오류 정정
=====================
런치패드에서는 다음 브라우저를 지원합니다. 

Mozilla Firefox 버전 3.5 이상
Microsoft(R) Internet Explorer, 버전 6.0 이상(Microsoft Windows(R) 플랫폼 전용)

런치패드가 시작되지 않는 오류를 정정하려면 다음 단계를 수행하십시오.


1.  지원되는 브라우저를 설치하거나 업데이트하십시오. 

    Linux(R) 및 UNIX(R) 플랫폼의 경우, 지원되는 브라우저의 위치를 내보내십시오.

    예를 들어 명령행에 다음을 입력하십시오.
    export BROWSER=/usr/bin/firefox

2. 제품 DVD에 액세스할 수 없는 경우 DVD를 DVD 드라이브에 삽입하십시오. 

3. Linux 및 UNIX 플랫폼의 경우, 필요한 경우에 드라이브를 마운트하십시오.

4.  런치패드를 다시 시작하십시오.
	Windows 플랫폼의 경우, 명령행에 다음을 입력하십시오.
		launchpad.exe
	Linux 및 UNIX 플랫폼의 경우 명령행에 다음을 입력하십시오.
		./launchpad.sh
