Correzione dell'errore

Il launchpad supporta i seguenti browser:

  • Mozilla Firefox versione 3.5 e successive
  • Microsoft® Internet Explorer, versione 6.0 e successive (solo piattaforme Microsoft Windows®)

Per correggere l'errore che impedisce l'avvio del launchpad, attenersi alla seguente procedura:

  1. Installare o aggiornare un browser supportato.

    In piattaforme Linux® e UNIX®, esportare l'ubicazione del browser supportato.
    Nella riga comandi, ad esempio, immettere:
    export BROWSER=/usr/bin/firefox

  2. Se il DVD del prodotto non è accessibile, inserire il DVD in un'unità DVD.
  3. Su piattaforme Linux e UNIX, montare la periferica se necessario.
  4. Riavviare il launchpad
    Su piattaforme Windows, nella riga comandi immettere:
    launchpad.exe
    Su piattaforme Linux e UNIX, nella riga comandi immettere:
    ./launchpad.sh
