更正錯誤
=====================
啟動程式支援下列瀏覽器： 

Mozilla Firefox 3.5 版以及更新版本
Microsoft® Internet Explorer 6.0 版以及更新版本（僅限 Microsoft Windows® 平台）

若要更正讓啟動程式無法啟動的錯誤，請遵循下列步驟： 

1.  安裝或更新為支援的瀏覽器。 

    在 Linux® 及 UNIX® 平台上，匯出支援的瀏覽器位置。 
    例如，在指令行輸入下列指令：
    export BROWSER=/usr/bin/firefox

2.  如果無法存取產品 DVD，請將 DVD 插入 DVD 光碟機。 

3.  必要的話，在 Linux 及 UNIX 平台上裝載磁碟機。

4.  重新啟動啟動程式
	在 Windows 平台的指令行上，輸入下列指令：
		launchpad.exe
	在 Linux 及 UNIX 平台的指令行上，輸入下列指令：
		./launchpad.sh
