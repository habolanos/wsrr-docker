エラーの訂正

ランチパッドでは以下のブラウザーをサポートします。

  • Mozilla Firefox バージョン 3.5 以降
  • Microsoft® Internet Explorer、バージョン 6.0 以降 (Microsoft Windows® プラットフォームのみ)

ランチパッドが開始しない原因となっているエラーを訂正するには、以下の手順を実行します。

  1. サポートされているブラウザーをインストールするか更新してください。

    Linux® および UNIX® プラットフォームの場合、サポートされるブラウザーの場所をエクスポートします。
    例えば、コマンド行に以下のように入力します。
    export BROWSER=/usr/bin/firefox

  2. 製品 DVD を利用できない場合、DVD ドライブに DVD を挿入します。
  3. Linux および UNIX プラットフォームの場合、必要に応じてドライブをマウントします。
  4. ランチパッドを再始動します。
    Windows プラットフォームの場合、コマンド行で以下のように入力します。
    launchpad.exe
    Linux および UNIX プラットフォームの場合、コマンド行で以下のように入力します。
    ./launchpad.sh
