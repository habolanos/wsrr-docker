Correcting the error~15761

The launchpad supports the following browsers: ~15762

  • Mozilla Firefox version 3.5 and later~15763
  • Microsoft® Internet Explorer, version 6.0 and later (Microsoft Windows® platforms only)~15764

To correct the error that prevents the launchpad from starting, follow these steps: ~15765

  1. Install or update to a supported browser. ~15766

    On Linux® and UNIX® platforms, export the location of the supported browser. ~15767
    For example, type the following at the command line: ~15768
    export BROWSER=/usr/bin/firefox ~15769

  2. If the product DVD is not accessible, insert the DVD into a DVD drive. ~15770
  3. On Linux and UNIX platforms, mount the drive if necessary. ~15771
  4. Restart the launchpad ~15772
    On Windows platforms, type the following at the command line:~15773
    launchpad.exe~15774
    On Linux and UNIX platforms, type the following at the command line:~15775
    ./launchpad.sh~15776
