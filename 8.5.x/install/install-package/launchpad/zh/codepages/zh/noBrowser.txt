更正错误
=====================
启动板支持下列浏览器： 

Mozilla Firefox V3.5 及更高版本
Microsoft(R) Internet Explorer V6.0 及更高版本（仅限于 Microsoft Windows(R) 平台）

要更正使启动板无法启动的错误，请遵循下列步骤： 

1.  安装或更新到受支持的浏览器。 

    在 Linux® 和 UNIX® 平台上，导出受支持浏览器的位置。 
    例如，在命令行上输入以下命令：
    export BROWSER=/usr/bin/firefox

2.  如果产品 DVD 不在 DVD 驱动器中，那么请插入产品 DVD。 

3.  在 Linux 和 UNIX 平台上，安装驱动器（如有需要）。

4.  将启动板重新启动
	在 Windows 平台上，请从命令行输入以下命令：
		launchpad.exe
	在 Linux 和 UNIX 平台上，请从命令行输入以下命令：
		./launchpad.sh
