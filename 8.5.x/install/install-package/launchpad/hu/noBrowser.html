A hiba kijavítása

Az indítópult a következő böngészőket támogatja:

  • Mozilla Firefox 3.5 és újabb változat
  • Microsoft® Internet Explorer 6.0 vagy újabb (csak Microsoft Windows® platformok esetén)

Az indítópult elindítását megakadályozó hiba kijavításához kövesse az alábbi lépéseket:

  1. Telepítsen egy támogatott böngészőt, vagy frissítsen egy támogatott böngészőre.

    Linux® és UNIX® platformokon exportálja a támogatott böngésző helyét.
    Például írja be az alábbiakat a parancssorba:
    export BROWSER=/usr/bin/firefox

  2. Ha a termék DVD lemeze nem érhető el, helyezze a DVD lemezt a DVD-meghajtóba.
  3. Linux és UNIX platformokon szükség esetén illessze be a meghajtót.
  4. Indítsa újra az indítópultot
    Windows platformokon írja be a következő parancsot a parancssorba:
    launchpad.exe
    Linux és UNIX platformokon írja be a következő parancsot a parancssorba:
    ./launchpad.sh
