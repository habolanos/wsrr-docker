Corectarea erorii
=====================
Launchpad-ul suportă următoarele browser-e: 

Mozilla Firefox versiunea 3.5 şi ulterioare
Microsoft(R) Internet Explorer, versiunea 6.0 şi ulterioare (doar platforme Microsoft Windows(R))

Pentru a corecta eroarea care împiedică pornirea launchpad-ului, urmaţi aceşti paşi: 

1.  Instalaţi sau actualizaţi la un browser suportat. 

    Pe platformele Linux(R) şi UNIX(R), exportaţi locaţia browser-ului suportat. 
    De exemplu, tastaţi următoarele la linia de comandă:
    export BROWSER=/usr/bin/firefox

2.  Dacă DVD-ul produsului nu este accesibil, introduceţi DVD-ul în unitatea DVD. 

3.  Pe platformele Linux şi UNIX, montaţi unitatea dacă este necesar.

4.  Reporniţi launchpad-ul
	Pe platformele Windows, tastaţi următoarele la linia de comandă:
		launchpad.exe
	Pe platformele Linux şi UNIX, tastaţi următoarele la linia de comandă:
		./launchpad.sh
