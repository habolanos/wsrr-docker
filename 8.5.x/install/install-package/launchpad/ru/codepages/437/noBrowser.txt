Исправление ошибки
=====================
Панель запуска поддерживает следующие браузеры: 

Mozilla Firefox версии 3.5 и более поздних версий,
Microsoft(R) Internet Explorer версии 6.0 и более поздних версий (только платформы Microsoft Windows(R))

Для исправления ошибки, из-за которой не была показана панель запуска, выполните следующие действия: 

1.  Установите или обновите браузер. 

    В Linux(R) и UNIX(R) следует экспортировать расположение поддерживаемого браузера. 
    Например, введите в командной строке:
    export BROWSER=/usr/bin/firefox

2.  Если диск DVD продукта был извлечен, то вставьте этот DVD в дисковод DVD. 

3.  На платформах Linux и UNIX смонтируйте привод при необходимости.

4.  Перезапустите программу панели запуска
	На платформах Windows необходимо открыть командную строку и ввести следующее:
		launchpad.exe
	На платформах Linux и UNIX необходимо открыть командную строку и ввести следующее:
		./launchpad.sh
