Correction de l'erreur

Le tableau de bord prend en charge les navigateurs suivants :

  • Mozilla Firefox version 3.5 et versions suivantes
  • Microsoft Internet Explorer, version 6.0 et versions ultérieures (plateformes Microsoft Windows uniquement)

Pour corriger l'erreur qui empêche le tableau de bord de démarrer, procédez comme suit :

  1. Installez ou mettez à jour un navigateur pris en charge.

    Sur une plateforme Linux ou UNIX, exportez l'emplacement du navigateur pris en charge.
    Par exemple, entrez la commande suivante :
    export BROWSER=/usr/bin/firefox

  2. Si le DVD du produit n'est pas accessible, insérez le DVD dans un lecteur de DVD.
  3. Sous Linux et UNIX, montez l'unité si nécessaire.
  4. Redémarrez le tableau de bord
    Sous Windows, entrez la commande suivante :
    launchpad.exe
    Sous Linux et UNIX, entrez la commande suivante :
    ./launchpad.sh
