Behebung des Fehlers

Dieses Launchpad unterstützt die folgenden Browser:

  • Mozilla Firefox ab Version 3.5
  • Microsoft® Internet Explorer ab Version 6.0 (nur Microsoft Windows®-Plattformen)

Führen Sie die folgenden Schritte aus, um den Fehler zu beheben, der den Start des Launchpads verhindert:

  1. Installieren Sie einen unterstützten Browser oder aktualisieren Sie Ihren Browser entsprechend.

    Auf Linux®- und UNIX®-Plattformen müssen Sie die Position des unterstützten Browsers exportieren.
    Geben Sie z. B. Folgendes in der Befehlszeile ein:
    export BROWSER=/usr/bin/firefox

  2. Sollte auf die Produkt-DVD nicht zugegriffen werden können, legen Sie diese DVD in ein DVD-Laufwerk ein.
  3. Gegebenenfalls müssen Sie auf Linux- und UNIX-Plattformen das Laufwerk anhängen.
  4. Starten Sie das Launchpad erneut
    Geben Sie auf Windows-Plattformen Folgendes in der Befehlszeile ein:
    launchpad.exe
    Geben Sie auf Linux- und UNIX-Plattformen Folgendes in der Befehlszeile ein:
    ./launchpad.sh
