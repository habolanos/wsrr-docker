ความต้องการของระบบ

สำหรับ ข้อมูลเกี่ยวกับความต้องการของฮาร์ดแวร์และซอฟต์แวร์สำหรับการติดตั้ง ผลิตภัณฑ์ โปรดดูที่ ความต้องการของระบบสำหรับ IBM Installation Manager

คำแนะนำในการติดตั้ง

สำหรับ คำแนะนำการติดตั้งแบบทีละขั้นตอน โปรดดูที่คู่มือการติดตั้ง สำหรับ Installation Manager หรือ ไปที่ศูนย์ข้อมูลสำหรับผลิตภัณฑ์ IBM ที่คุณกำลังติดตั้ง

สำหรับคำแนะนำเกี่ยวกับการติดตั้ง ชุดซ่อมแซมนี้ไม่ว่าจะเป็นการอัพเดต หรือในเวลาเดียวกับที่คุณติดตั้ง ผลิตภัณฑ์พื้นฐาน โปรดดูที่ดาวน์โหลด IBM Installation Manager

ปัญหาที่พบ

ปัญหาที่พบ จะถูกบันทึกในเอกสารแต่ละฉบับในฐานความรู้ของการสนับสนุน http://www.ibm.com/software/rational/support/ เมื่อปัญหาถูกค้นพบ และได้รับการแก้ไข ฐานความรู้จะถูกอัพเดต และบำรุงรักษาด้วยข้อมูลใหม่ โดย การค้นหาฐานความรู้ คุณจะสามารถค้นหาวิธีแก้ไข หรือโซลูชัน สำหรับปัญหาได้อย่างรวดเร็ว

การสนับสนุนซอฟต์แวร์IBM Rational

สำหรับข้อมูลการติดต่อ และแนวทาง หรือเอกสารอ้างอิง ที่คุณจำเป็นต้องใช้เมื่อคุณต้องการการสนับสนุน โปรดอ่านที่คู่มือการสนับสนุนซอฟต์แวร์ IBM

สำหรับ FAQ รายการของปัญหาที่พบ และ การแก้ไข และข้อมูลการสนับสนุนอื่นๆ โปรดไปที่พอร์ทัลการสนับสนุน IBM Installation Manager

สำหรับข่าว กิจกรรม และข้อมูลอื่นๆ ที่เกี่ยวกับผลิตภัณฑ์ โปรดไปที่เว็บไซต์ของซอฟต์แวร์ IBM Rational

ก่อนที่คุณจะติดต่อฝ่ายสนับสนุนซอฟต์แวร์ IBM Rational ให้รวบรวมข้อมูลพื้นหลังที่คุณต้องใช้เพื่ออธิบาย ปัญหาของคุณ เมื่ออธิบายปัญหาให้กับผู้เชี่ยวชาญด้านการสนับสนุนซอฟต์แวร์ของ IBM ให้ระบุถึงปัญหาให้ชัดเจนมากที่สุด และอธิบายข้อมูลพื้นหลังที่เกี่ยวข้อง ทั้งหมดเพื่อที่ผู้เชี่ยวชาญจะสามารถช่วยคุณแก้ไขปัญหา ได้อย่างมีประสิทธิภาพ เพื่อเป็นการประหยัดเวลา ควรหาคำตอบเกี่ยวกับคำถามเหล่านี้ไว้ล่วงหน้า:

  • เวอร์ชันของซอฟต์แวร์ที่คุณใช้รันเมื่อเกิดปัญหาคือเวอร์ชันอะไร
  • คุณมีไฟล์บันทึก, เธรด หรือข้อความที่เกี่ยวข้องกับ ปัญหาหรือไม่
  • คุณสามารถทำให้เกิดปัญหาอีกครั้งได้หรือไม่ หากทำได้ ขั้นตอนใดที่คุณสามารถใช้เพื่อ ทำให้เกิดปัญหาขึ้นอีกครั้ง
  • มีวิธีแก้ปัญหาหรือไม่
  • หากมี ให้เตรียมเพื่ออธิบายวิธีแก้ปัญหา
