Известные неполадки

Известные неполадки описаны в отдельных документах в Базе знаний службы поддержки. По мере обнаружения и устранения неполадок база знаний обновляется и пополняется новой информацией. Выполняя поиск в базе знаний, можно быстро найти обходные пути и способы устранения неполадок.

Поддержка программного обеспечения IBM Rational

Контактная информация и рекомендации или справочные материалы, необходимые при обращении в службу поддержки, приведены в документе Руководство по поддержке программного обеспечения IBM.

Часто задаваемые вопросы, списки известных проблем и их исправлений, а также другая информация службы поддержки приведены на веб-страницеПортал службы поддержки IBM Installation Manager.

Новости о продукте, события и другая информация приведены на веб-сайте Программное обеспечение IBM Rational.

Перед обращением в службу поддержки программного обеспечения IBM Rational соберите информацию, требуемую для описания неполадки. При описании неполадки специалисту службы поддержки IBM будьте максимально конкретны и укажите всю связанную информацию, чтобы специалист мог помочь вам наиболее эффективно. Для экономии времени требуется знать ответы на следующие вопросы:

  • Версии ПО, работавшего во время неполадки?
  • Существуют ли протоколы, трассировки или сообщения, связанные с неполадкой?
  • Можно ли воспроизвести неполадку? Если да, то какие действия для этого предпринимались?
  • Существует ли способ обхода неполадки?
  • Если да, будьте готовы описать способ обхода.
